== Description ==

Used by many travelers worldwide, iVisa Travel is the best tool to know which countries need a travel visa.

In addition, through iVisa Travel travels can process visas for many of those countries using a 100% online and simplified application process.

If you are a travel related blogger you can also use this plugin to earn a monthly recurring revenue through iVisa's affiliate program. You can learn about our affiliate program on: [https://www.ivisa.com/affiliates](https://www.ivisa.com/affiliates)

To get started: activate the iVisa Travel plugin and then go to your iVisa Travel Settings page to set up the plugin.


== Installation ==

1. Copy the `ivisa` folder into to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings-> iVisa Travel screen to configure the plugin


== Frequently Asked Questions ==

= Do I need to update anything on my end?

No, the travel widget checks against iVisa database to provide the most up to date information regarding travel visas worldwide.

= Is this widget affiliated with any Government? =

No, [iVisa.com](www.ivisa.com) is a private company making the travel visa process as simple as it can be.
